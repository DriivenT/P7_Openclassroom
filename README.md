
### Installing

1. Cloner le repo
2. Lancer ```npm install```
3. Configurer ```config/config.json```
4. Lancer ```sequelize db:create && sequelize db:migrate```
5. // Import sauvegarde.sql // ?
6. Créer un fichier ".env" pour l'utilisation de dotenv contenant:
    - HOST = localhost
    - PORT = 3307
    - USER = admin
    - PASSWORD = admin
    - DB = groupomania
    - HEADERS = *
